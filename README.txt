INTRODUCTION
------------

This module exposes base fields on the Manage Display screen so you can
rearrange and format them just like any other field.

 * For a full description of the module, visit the project page:
   https://www.drupal.org/sandbox/yesthatdarya/3038983

 * To submit bug reports and feature suggestions, or to track changes:
   https://www.drupal.org/project/issues/3038983


REQUIREMENTS
------------

No special requirements.


INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module.
   See: https://www.drupal.org/node/895232 for further information.


CONFIGURATION
-------------

 * Go to Administration » Configuration » User interface » Base field
   display settings to select which fields can be managed through the
   Manage Display UI.


TROUBLESHOOTING
---------------

 * If fields appear twice on the front end, you likely need to modify your
   theme's Twig files.
